A social networking platform designed to connect people by their shared interests and passions.
We understand that having a shared interest is one of the most effective ways to connect with others. That's why we've built a platform that allows users to join groups based on their hobbies, connect with like-minded individuals, and plan events to enjoy their hobbies together. Whether you're into sports, art, music, cooking, gaming, or any other hobby, you'll be able to find a group of people who share your interests and are excited to connect.

*Hobby Hive is currently under development. Exciting new features are on the horizon. Stay tuned for updates!*

- Websockets

- SCSS/CSS
- SQL
- MUI
- Ant Design


## Core Features
- Seamless Login: Utilize the convenience and security of your Google, LinkedIn, or Facebook accounts to effortlessly access Hobby Hive.
- Instant Messaging: Engage in real-time conversations with fellow group members through our intuitive chat interface.
- Discover and Connect: Effortlessly explore and connect with groups that align with your interests, using our comprehensive search feature.
- Stay Organized: Stay informed about upcoming events hosted by your group through our user-friendly calendar integration.

## Features in Progress

## How to Contribute
1. Clone the repo and make a new branch
2. Run ```npm install``` in your terminal
3. Set up your Auth0 environment. You can refer to the docs <a href='https://auth0.com/docs/get-started'>here</a>.
4. Run ```npm run start``` in your terminal.
5. Run ```npm run server``` and ```npm run websocketserver``` to spin up the server. 
7. Navigate to [localhost:3000](http://localhost:3000). 
8. Add a feature, fix a bug or refactor some code
9. Write/update tests for the changes you made, if necessary
10. Open a Pull Request with a comprehensive description of changes to the dev branch
